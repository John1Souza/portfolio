Como estudante de Análise e Desenvolvimento de Sistemas, estou em constante evolução no mundo da tecnologia, sempre buscando novas formas de aprimorar minhas habilidades e contribuir para projetos inovadores. Minha experiência abrange desde o desenvolvimento web, utilizando HTML, CSS, JavaScript e frameworks como React, até o backend, onde atuo com Java e Spring, além de gerenciar bancos de dados relacionais como SQL e PostgreSQL, e não relacionais, como MongoDB. Essa base sólida me permite abordar projetos com uma visão completa, do front ao back-end, entregando soluções eficientes e de alto impacto.

Durante esse período, desenvolvi não apenas habilidades técnicas, mas também um olhar estratégico para solucionar problemas e atender às demandas de forma ágil e assertiva. Estou pronto para encarar novos desafios e somar minha experiência e paixão por tecnologia a iniciativas que façam a diferença.

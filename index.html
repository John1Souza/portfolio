Sobre Mim

Olá! Sou um desenvolvedor apaixonado com profundo interesse em tecnologia e tudo relacionado à computação. Ao longo dos 2 últimos anos, dediquei-me a adquirir conhecimento com relação às áreas de tecnologia, motivado pela minha paixão por aprender e explorar novas tecnologias.

Sou proficiente em desenvolvimento full-stack, com foco particular em React e NextJS para desenvolvimento web. No lado Back-End, tenho experiência com ferramentas como Spring Boot para criar APIs RESTful, junto com Spring Data JPA para integração com bancos de dados como MySQL, PostgreSQL ou MongoDB. Hibernate (ORM) e Maven para gerenciar dependências. Ferramentas como Git para versionamento de código, enquanto JUnit e Mockito para os testes, Postman para testar APIs e o básico de Docker para conteinerização — habilidades que venho aprimorando por meio de projetos pessoais e experiência profissional.

Atualmente estou cursando um curso de Análise e Desenvolvimento de Sistemas, estágio em andamento na EBSERH - Empresa Brasileira de Serviços Hospitalares.
